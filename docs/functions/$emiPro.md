---
title: $emiPro
description: $emiPro will automatically make emi pro canvas.
id: emiPro
---

It creates new emi pro canvas.

## Usage

```php
$emiPro
```

## Parameters

None

## Example(s)

It will send new ayaka pro canvas.

```js
bot.command({
    name: "emi",
    code: `
    $sendCanvas[emiPro;msg]
    $emiPro
    `
});
```
