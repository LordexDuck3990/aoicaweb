It draws lines canvas. (in path)

## Usage

```php
$drawLines[canvas;...waypoints]
```

## Parameters

| Field | Type | Description | Required |
| ----- | ---- | ----------- | :------: |
| canvas | string | Name of the canvas. | true |
| waypoints | (startX;startY;endX;endY;...) | Waypoints. | true |
