id: measureText
---

It returns text size.

## Usage

```php
$measureText[canvas;text;argument]
```

## Parameters

| Field | Type | Description | Required |
| ----- | ---- | ----------- | :------: |
| canvas | string | Name of the canvas. | true |
| text | string | Text. | true |
| argument | string | Size argument to return. (width/height) | true |
