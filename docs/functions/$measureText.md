# $measureText

It returns text size.

---

## Usage

```
$measureText[canvas;text;argument]
```

## Parameters

| Field | Type | Description | Required |
| ----- | ---- | ----------- | :------: |
| canvas | string | Name of the canvas. | yes |
| text | string | Text. | yes |
| argument | string | Size argument to return. (width/height) | yes |
