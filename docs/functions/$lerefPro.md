# $lerefPro

It creates new leref pro canvas.

---

## Usage

```
$lerefPro
```

## Parameters

None

## Example(s)

It will send new leref pro canvas.

```js
bot.command({
    name: "leref",
    code: `
    $sendCanvas[lerefPro;msg]
    $lerefPro
    `
});
```
