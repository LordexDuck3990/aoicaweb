It starts path.

## Usage

```php
$startPath[canvas;x;y]
```

## Parameters

| Field | Type | Description | Required |
| ----- | ---- | ----------- | :------: |
| canvas | string | Name of the canvas. | true |
| x | number | X position. | true |
| y | number | Y position. | true |
