# $ayakaPro

It creates new leref pro canvas.

---

## Usage

```
$ayakaPro
```

## Parameters

None

## Example(s)

It will send new leref pro canvas.

```js
bot.command({
    name: "ayaka",
    code: `
    $sendCanvas[ayakaPro;msg]
    $ayakaPro
    `
});
```
