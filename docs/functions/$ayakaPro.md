title: $ayakaPro

id: ayakaPro
---

It creates new ayaka pro canvas.

## Usage

```php
$ayakaPro
```

## Parameters

None

## Example(s)

It will send new ayaka pro canvas.

```js
bot.command({
    name: "ayaka",
    code: `
    $sendCanvas[ayakaPro;msg]
    $ayakaPro
    `
});
```
