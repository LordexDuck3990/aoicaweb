# $canvasSize

It returns canvas size.

---

## Usage

```
$canvasSize[canvas;argument]
```

## Parameters

| Field | Type | Description | Required |
| ----- | ---- | ----------- | :------: |
| canvas | string | Name of the canvas. | yes |
| argument | string | Size argument to return. (width/height) | yes |
